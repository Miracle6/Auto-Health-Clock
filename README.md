# Auto-Health-Clock
自动打卡。马房山男子职业技术学校。Node.js。带前端界面登录。

### 使用说明

1. 安装最新版Node

   直接去官网 [Node.js (nodejs.org)](https://nodejs.org/zh-cn/)

2. 下载本代码文件夹

3. `cmd`切换到文件夹根目录，运行`npm i` 安装程序。

4. `node app.js`即可以开启服务，打开`localhost:666`

5. 即可测试打卡流程。
